Using keyd in Wayland to rebind your right alt key to escape (linux)

I recently started messing around with hyprland (which uses wayland) on my archcraft linux installation. Unlike i3 (which uses x11) my xmodmap just isn’t built for it.

if you’re on arch linux, you can easily run the command:

sudo pacman -S keyd

On my system I found keyd here:

/etc/keyd/

I then create a default config file using vim, but feel free to use nano or your text editor of choice. Just make sure to create it using sudo priviledges.

sudo vim /etc/keyd/default.conf

file looks like: 

                

                    [ids]

                    *

                    [main]

                    #remaps right alt to escape
                    rightalt = escape

now you want to enable keyd. You can open up a terminal and use the command:

sudo systemctl enable keyd
